yMurMur

yMurMur это Web-обертка над программой Murka 1.4.1.
yMurMur is a Web-wrapper for Murka 1.4.1.

Руководство:

  1. Откройте MyMcGee Y-Utility
  2. Поставьте галочку напротив пункта "Generate Fluxus phylogenetic network .ych data"
  3. Вставьте набор гаплотипов в поле "Paste haplotype or setup data here:"
  4. Нажмите кнопку "Execute"
  5. В открывшемся окне "Y-DNA Comparison Tables" скопируйте содержимое поля "Fluxus data - paste into a .ych file"
  6. Вставьте его в поле "Fluxus (.YCH)" на текущей странице
  7. Нажмите кнопку "Send Request"

Manual:

  1. Open MyMcGee Y-Utility
  2. Check the box next to "Generate Fluxus phylogenetic network .ych data"
  3. Paste the haplotype set in the "Paste haplotype or setup data here:" field
  4. Click the "Execute" button
  5. In the "Y-DNA Comparison Tables" window that opens, copy the contents of the "Fluxus data - paste into a .ych file" field
  6. Paste it into the field "Fluxus (.YCH)" on the current page
  7. Click the "Send Request" button

Предупреждение: подготовка ответа может занять до 10 секунд. Загрузка ZIP-файла начнется автоматически. Дистанции в файлах указаны в поколениях.
Warning: it may take up to 10 seconds to prepare a response. The download of the ZIP file will start automatically. The distances in the files are given in generations.
